## Oyun Nasıl Oynanır?
1. Bilgisayar 1-100 arası bir yıldız seçer
2. Oyuncular sırayla sayı tahmin eder
3. Yıldızı ilk bulan puan kazanır
4. 3 zorluk seviyesi var
